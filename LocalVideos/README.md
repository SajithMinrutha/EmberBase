# EmberStream

A local video library for organizing tuition classes by subject and unit.
Drop videos into folders and the index updates automatically while the server runs.

## Folder structure

```
Videos/
  Chemistry/
    Unit 01/
    Unit 02/
  Maths/
    Unit 01/
    Unit 02/
```

Videos placed directly under a subject folder appear in a "General" unit.

## Update videos

```bash
node generate-videos.js
```

### Auto-update (watch mode)

```bash
node generate-videos.js --watch
```

## Run locally

```bash
node serve.js
```

Then open `http://localhost:5175`.

### Disable auto-watch

```bash
WATCH_VIDEOS=0 node serve.js
```

## Notes

- `videos-data.js` is generated by `generate-videos.js`.
- Supported extensions: `.mp4`, `.m4v`, `.webm`, `.mov`, `.mkv`.
